Excepciones de horarios

Crea periodos especiales en los que los retrasos no se contabilizan, por ejemplo vacaciones, semanas institucionales o permisos generales.

{% if request.user.role == "admin" %} Como administrador puedes crear excepciones globales o por dependencia. {% else %} Como líder solo puedes crear excepciones para {{ request.user.get_department_display }}. {% endif %}
Estados de las excepciones:
Vigente: la excepción está activa y la fecha actual está dentro del rango.
Programada: la excepción está activa, pero su fecha inicial todavía no llega.
Activa: la excepción sigue marcada como activa, aunque su rango de fechas ya terminó.
Inactiva: la excepción está desactivada y no aplica sobre las sesiones.
